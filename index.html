How to Use Enter a CU Boulder identifier, for example,

  • course number (e.g., CSCI 3010)
  • instructor (e.g., Sreesha Nath)
  • department (e.g., Computer Science)
  • college (e.g., Engineering)

then press Submit. For added functionality, create an account and store schedules.

About This is a term project for CU Boulder's Intensive Programming Workshop course. It's a program I've been wanting to build for a while after inspiration by a former roommate, Brandon, who I credit with my interest in programming.
